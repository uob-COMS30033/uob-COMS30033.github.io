Installation Instructions:

Jupiter Notebook - For all COMS30035 needs you are encouraged to install Anaconda (Python 3.7) as it bundles all the course's requirements. Alternatively for manual installation, you will require Python 3.7.x with 'Jupyter' and 'iPython' both possibly in version 4.x.x as well as the following packages: 'Pillow', 'scikit-image', 'matplotlib', 'numpy', 'scipy', 'scikit', 'pygments' and 'scikit-learn'

In the lab Linux machines, you should be able to just run Jupyter Notebook with this command
$ /opt/anaconda3-4.4.0/bin/jupyter notebook
